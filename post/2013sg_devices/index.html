Mapping von SG-Devices anzeigen

June 10, 2013

Um die Zuweisung von physischen SCSI-Geräten zu logischen devices darzustellen, gibt es ein simples Tool namens _sgmap. Das Binary befindet sich im Paket _sg3utils

<br /> yum install sg3_utils<br />

Dann liefert das Proramm die Zuweisungsliste:

`

sg_map

/dev/sg0 /dev/sda
/dev/sg1 /dev/scd0
/dev/sg2 /dev/scd1
/dev/sg3 /dev/sdb
/dev/sg4 /dev/sdc
/dev/sg5 /dev/sdd
/dev/sg6
/dev/sg7 /dev/nst0
/dev/sg8
/dev/sg9 /dev/sde
/dev/sg10 /dev/sdf
/dev/sg11
/dev/sg12 /dev/sdg
/dev/sg13
`
